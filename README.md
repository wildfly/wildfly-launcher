WildFly Launcher
============
https://wildfly.org

This project provides a tool than can be used to launch various types of WildFly processes.

Building
-------------------

Prerequisites:

* JDK 11 or newer - check `java -version`
* Maven 3.6.0 or newer - check `mvn -v`

To build with your own Maven installation:

> mvn install
